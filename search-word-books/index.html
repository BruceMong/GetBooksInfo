Retrieve all books (title, author(s), date) that contain a specific word, then export them to PDF or Excel.

×
  1. Go to the Google Cloud Platform development console page: https://console.cloud.google.com/
  2. Log in to your Google account or create one if you don't have one.
  3. Click on the "Select a project" button at the top of the page, then click on the "New project" button to create a new project.
  4. Give your project a name and click "Create."
  5. Once your project is created, click on the menu icon in the top left of the page, then click on "APIs & Services."
  6. Click on the "Library" button to search for the Google Books API. Find the Google Books API and click "Enable."
  7. Once the API is enabled, click on the "Create Credentials" button to create an API key.
  8. Select "API key" and click "Create" to generate your API key.
  9. Copy your API key and paste it in the Google Books API input.
